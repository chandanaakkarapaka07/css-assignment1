What is CSS

CSS stands for Cascading Style Sheets. It is a style sheet language which is used to describe the look and formatting of a document written in markup language. It provides an additional feature to HTML. It is generally used with HTML to change the style of web pages and user interfaces. It can also be used with any kind of XML documents including plain XML, SVG and XUL.

CSS Id Selector

The id selector selects the id attribute of an HTML element to select a specific element. An id is always unique within the page so it is chosen to select a single, unique element. It is written with the hash character (#), followed by the id of the element.

CSS Class Selector

The class selector selects HTML elements with a specific class attribute. It is used with a period character . (full stop symbol) followed by the class name.

I. Introduction

Cascading Style Sheets (CSS) is a powerful language used for styling and formatting web documents. CSS selectors play a crucial role in applying styles to specific elements, classes, and IDs within HTML documents. This report aims to provide a comprehensive overview of CSS selectors and their usage based on identified elements, classes, and IDs.

II. Basic CSS Selectors

  1. Universal Selector (*)
    • The universal selector targets all elements on a webpage.
    • Usage example: * { margin: 0; padding: 0; } resets margins and padding for all elements.
  2. Element Selector
    • Targets specific HTML elements.
    • Usage example: p { color: blue; } sets the text color of all paragraphs to blue.
  3. Class Selector (.)
    • Targets elements with a specific class attribute.
    • Usage example: .highlight { font-weight: bold; } makes text with the "highlight" class bold.
  4. ID Selector (#)
    • Targets elements with a specific class attribute.
    • Usage example: .highlight { font-weight: bold; } makes text with the "highlight" class bold.

III. Advanced CSS Selectors

  1. Descendant Selector ( )
    1. Targets nested elements.
    2. Usage example: article p { font-style: italic; } italicizes paragraphs within an article.
  2. Child Selector (>)
    1. Targets direct children of a parent element.
    2. Usage example: ul > li { list-style-type: square; } styles only direct list items of a list.
  3. Adjacent Sibling Selector (+)
    1. Targets an element that is immediately preceded by a specific element.
    2. Usage example: h2 + p { font-size: 1.2em; } increases the font size of paragraphs directly following an h2.
